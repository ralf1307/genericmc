MULTIMC(1)
==========
:doctype: manpage


NAME
----
genericmc - a launcher and instance manager for Minecraft.


SYNOPSIS
--------
*genericmc* ['OPTIONS']


DESCRIPTION
-----------
GenericMC is a custom launcher for Minecraft that allows you to easily manage
multiple installations of Minecraft at once. It also allows you to easily
install and remove mods by simply dragging and dropping.
Here are the current features of MultiMC.

OPTIONS
-------
*-d, --dir*='DIRECTORY'::
    Use 'DIRECTORY' as the GenericMC root.

*-l, --launch*='INSTANCE_ID'::
    Launch the instance specified by 'INSTANCE_ID'.

*--alive*::
    Write a small 'live.check' file after GenericMC starts.

*-h, --help*::
    Display help text and exit.

*-v, --version*::
    Display program version and exit.

EXIT STATUS
-----------
*0*::
    Success

*1*::
    Failure (syntax or usage error; configuration error; unexpected error).
